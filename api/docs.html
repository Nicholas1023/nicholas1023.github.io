API Documentation

Base URL: https://nicholas1023.github.io/api/v1

Endpoints

Method: GET
  • / Retrives a full list of projects.
  • /id1 Retrives project with ID 1.
  • /id2 Retrives project with ID 2.
  • /id3 Retrives project with ID 3.
  • /id4 Retrives project with ID 4.

Example Response

<?xml version="1.0" encoding="UTF-8"?>
<content xmlns="http://nicholas1023.github.io">
    <project>
        <id>Project ID</id>
        <name>Project name</name>
        <url>Repository or website URL</url>
        <desc>Short description about project</desc>
    </project>
</content>
