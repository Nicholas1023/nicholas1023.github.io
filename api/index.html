Nicholas1023 API Documentation

The Nicholas1023 API provides information on my projects, for example, names and project websites in XML format.

Base URL: https://nicholas1023.github.io/api/v1/

Endpoints: 1, 2
Example endpoint URL: https://nicholas1023.github.io/api/v1/1

Response format:

<?xml version="1.0" encoding="UTF-8"?>
<data xmlns="http://nicholas1023.github.io/api">
    <status>HTTP status code</status>
    <name>Project name</name>
    <n>Project number</n>
    <l>Project website</l>
</data>
